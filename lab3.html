Hello, this is the page for lab 3, playing with mapbox.
As you can see the basic editor used for the first map allows for typical use (plus fun little novelties like changing line colour and width based on zoom levels).
However it is limited both in added functionality (i.e. the click to zoom and distance measures of the second map), and was klunky to avoid multiple labels of the same route when zoomed in close (requiring multiple label layers that each filtered out the text from other route names and were set to  have a high boundary area preventing multiple labels from the same layer)
